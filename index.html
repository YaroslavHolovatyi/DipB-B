Beer & Beverages

Mobile rebuild of the diploma project "Довідка пабів м. Львів" on a new stack: React Native + Expo, FastAPI on Python 3.12, PostgreSQL 16 with PostGIS, an AI Dungeon Master powered by OpenAI GPT-4o, and a WebSocket-based real-time layer for multi-device features.

Everything to install on your Ubuntu/Debian laptop before writing code — system packages, Docker, Python 3.12 via uv, Node 22 via nvm, Expo Go, VS Code, GitHub, OpenAI account, and a verify script that checks it all.

Database Schema

PostgreSQL 16 schema with PostGIS, citext, full-text search, JSONB. 44 tables grouped by domain — auth, bars, social, raids, checks, chat, achievements, Tavern Tales — with the design rationale for every column.
